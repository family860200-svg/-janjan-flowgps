# 阿文 — 工具清單

## 核心工具
| 工具 | 用途 | 使用頻率 |
|------|------|---------|
| Read | 讀取對話摘要、取得素材 | 高 |
| Write / Edit | 寫入草稿到指定路徑 | 高 |
| Skill（threads）| 生成 Threads 貼文草稿 | 高 |

## Skill 對照
| Skill | 用途 | 觸發時機 |
|------|------|---------|
| /threads | Threads 草稿（L3 鏈式推理）| JANJAN 說「發文」或阿助掃描到素材 |
| /ak-threads-booster | Threads 成長作業系統（AK 版）| 選題、草稿、分析、預測、審閱、追蹤器全套；需要策略性成長時使用 |

> **兩個 Skill 的差異**：
> - `/threads`：快速生成單篇草稿，有素材就動
> - `/ak-threads-booster`：基於 JANJAN 自己的貼文歷史，做選題策略、成效分析、帳號成長規劃——需要時間投入，但效果更系統

## 草稿存放規則
- 暫存：對話摘要末尾 `## 📝 Threads 草稿`
- 確定發出：移至 `F｜行動聚焦漏斗/Threads草稿/`（若有建立）

## 禁止使用
- ✗ 不主動發文到任何平台（需 JANJAN 確認）
- ✗ 不在沒有真實素材的情況下憑空創作

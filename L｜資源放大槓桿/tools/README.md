# Tools 導覽

這裡集中放置 FlowGPS 的獨立工具頁，依功能分成四類。

## 分類目錄

- `dashboard/`：儀表板與入口工具
- `complaint/`：客訴相關工具
- `marking/`：嘜頭與標示資料工具
- `container/`：貨櫃裝載與模擬工具

## 主要工具連結

- `dashboard/工具儀表板.html`
- `complaint/客訴處理機器人.html`
- `marking/嘜頭資料產生器.html`
- `container/貨櫃裝載模擬.html`

## 相容說明

為了避免舊連結失效，`tools/` 根目錄仍保留同名轉址頁：

- `tools/工具儀表板.html` -> `tools/dashboard/工具儀表板.html`
- `tools/客訴處理機器人.html` -> `tools/complaint/客訴處理機器人.html`
- `tools/嘜頭資料產生器.html` -> `tools/marking/嘜頭資料產生器.html`
- `tools/貨櫃裝載模擬.html` -> `tools/container/貨櫃裝載模擬.html`

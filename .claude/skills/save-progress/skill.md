# 存檔 Skill（save-progress）

> 對話結束時說「存檔」「結束」「掰掰」即自動執行

---

## 存檔流程

### Step 1：讀取隨手一記

讀取 `F｜行動聚焦漏斗/manual_notes.json` 當天的記錄，納入對話摘要的「✏️ 隨手一記」區塊。

### Step 2：生成對話摘要

整理今天對話重點，存到：

```
F｜行動聚焦漏斗/對話摘要/YYMMDD_主題.md
```

格式包含：
- ✅ 完成了什麼
- ❌ 還沒完成什麼
- 📍 下一步行動
- ✏️ 隨手一記（來自 manual_notes.json）
- 💰 八大財富帳戶加分分析

### Step 3：更新待辦清單

把「❌ 未完成」和「📍 下一步」寫入：

```
F｜行動聚焦漏斗/玩家待辦任務.md
```

### Step 4：更新遊戲進度快照

更新 `F｜行動聚焦漏斗/game-progress.md`，讓下次對話的 AI 快速掌握狀態：
- 主線任務進度
- 21 天習慣計畫天數
- 今日 Boss 完成狀況
- 能量狀態

### Step 5：Git 備份推上 GitHub

```bash
git add .
git commit -m "YYMMDD 時段存檔"
git push
```

push 完成後，**必須明確回報以下訊息**：

```
✅ 已推上 GitHub
📌 最新 commit：[commit hash] [commit message]
🕐 時間：[HH:MM]
```

若 push 失敗，**必須明確告知失敗原因**，不可靜默跳過。

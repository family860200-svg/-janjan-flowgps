## 對話 SOP

### 對話開始時
1. 執行 `date` 確認系統時間
2. 讀取 `F｜行動聚焦漏斗/對話摘要/` 最新摘要，了解上次做到哪裡
3. 讀取 `F｜行動聚焦漏斗/玩家待辦任務.md`，了解有哪些待辦
4. 簡短報告：上次做了什麼、今天有什麼待辦
5. 將待辦按**艾森豪矩陣**四象限分類報告：
   - 🔴 重要／緊急（立刻處理）
   - 🟡 重要／不緊急（排時間做）
   - 🟠 不重要／緊急（快速處理或委派）
   - ⚪ 不重要／不緊急（刪除或擱置）
6. 問今天想做什麼

### 換設備時（說「我回來了」時執行）
1. 執行 `git pull` 拉取最新版本
2. 讀取最新對話摘要，了解上次做到哪裡
3. 讀取待辦清單
4. 簡短報告現在進度，問要繼續什麼

### 對話結束時（說「存檔」「結束」「掰掰」時執行）
1. 整理今天對話重點，存到 `F｜行動聚焦漏斗/對話摘要/YYMMDD_主題.md`
2. 把未完成的待辦寫入 `F｜行動聚焦漏斗/玩家待辦任務.md`
3. 分析這次對話為哪些八大財富帳戶加了分
4. 用 git 存檔並 push 到 GitHub

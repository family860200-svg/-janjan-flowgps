# 配抄知識文件｜阿娟二號 NotebookLM 教練用

> 這份文件是給 NotebookLM 的訓練資料，用來建立「配抄教練」。
> 上傳後，AI 教練能用蘇格拉底式提問引導配抄決策。

---

## 一、什麼是配抄？

配抄是**紙廠業務的核心計算工作**。

工廠生產的紙是捲筒，要依照客戶的訂單尺寸，切割成一個個「平版」或是直接捲成捲筒出貨，就二種大類，

**關鍵概念：**

- **捲筒（Roll）**：一支出貨單位，每支由 3～5 個孔組成
- **孔（Hole）**：一個裁切寬度，單位是台尺
- **訂單孔（W / Want）**：客戶要的尺寸
- **庫存孔（P / Pad）**：用庫存寬度補滿一支捲筒的空位
- **台尺**：1 台尺 ≈ 30.3 公分

---

## 二、機台限制

每支捲筒的總寬度必須符合機台設定：

| 參數     | 意義                     | 範例值   |
| -------- | ------------------------ | -------- |
| 最小總寬 | 所有孔加起來不能低於此值 | 8.5 台尺 |
| 最大總寬 | 所有孔加起來不能超過此值 | 9.5 台尺 |
| 每孔最小 | 單一孔不能小於此值       | 1.6 台尺 |
| 每孔最大 | 單一孔不能大於此值       | 4.0 台尺 |

---

## 三、演算法邏輯（優先順序）

配抄的目標是：**在滿足機台限制的前提下，把所有訂單孔都配出去，同時最大化每支捲筒的訂單孔數量。**

### 優先順序

**第一優先：純訂單捲筒**

- 全部用訂單孔，不補庫存
- 孔數順序：先試 3 孔 → 4 孔 → 5 孔
- 能純訂單就不用庫存

**第二優先：1 孔庫存**

- 訂單孔 + 1 孔庫存補滿
- 最大化訂單孔數：先試 4 訂 +1 庫 → 3 訂 +1 庫 → 2 訂 +1 庫

**第三優先：2 孔庫存**

- 訂單孔 + 2 孔庫存補滿
- 最大化訂單孔數：先試 3 訂 +2 庫 → 2 訂 +2 庫 → 1 訂 +2 庫

**最後手段：單孔強制出**

- 當一個訂單孔湊不出任何合法捲筒，強制單孔出，附警示標記 ⚠️

---

## 四、庫存孔來源

庫存孔從「黃金尺寸」清單中選取，這些是工廠常備的庫存寬度：

```
1.6, 1.7, 1.8, 1.9, 2.0, 2.1, 2.2, 2.3, 2.4, 2.5,
2.6, 2.7, 2.8, 2.9, 3.0, 3.1, 3.2, 3.3, 3.4, 3.5,
3.6, 3.7, 3.8, 3.9, 4.0
```

---

## 五、四種計算版本

| 版本   | 單位         | 適用情境     |
| ------ | ------------ | ------------ |
| 台尺版 | 台尺         | 本地訂單標準 |
| MM 版  | 公釐（mm）   | 部分外銷訂單 |
| 公分版 | 公分（cm）   | 特定客戶規格 |
| 英吋版 | 英吋（inch） | 外銷英制規格 |

各版本演算法相同，只差單位換算。

## 六、輸出格式

計算完成後輸出：

```
PM1 配抄排程（YYYY/MM/DD）
機台限制：8.5～9.5 台尺 ／ 每孔 1.6～4.0 台尺

序號  孔1    孔2    孔3    孔4    孔5    總寬   粒數  說明
1     3.2W   2.9W   3.1W   —      —      9.2    3    全訂單
2     2.5W   2.4W   2.3W   2.1P   —      9.3    2    含庫存×1
...
```

- W = 訂單孔（綠色）
- P = 庫存孔（橘色）
- ⚠️ = 無法配對，強制單孔出

---

## 七、常見情境與判斷

**情境 1：訂單很多相近尺寸**
→ 容易湊出純訂單捲筒，效率最高

**情境 2：訂單尺寸很分散**
→ 需要大量庫存孔補滿，要注意庫存是否夠用

**情境 3：有單孔出現 ⚠️ 警示**
→ 代表這個訂單孔沒辦法跟任何組合配對，需要業務判斷：

- 是否接受這個尺寸單獨出一支？
- 還是請客戶調整數量？

**情境 4：客戶要求改單**
→ 重新輸入新的訂單資料，重新計算

---

## 八、業務判斷要點

配抄工具給的是**數學最佳解**，但業務還需要考慮：

1. **庫存是否真的有貨**——黃金清單是標準備貨，但特殊時期可能不足
2. **客戶優先順序**——急單優先配，一般單可以等
3. **損耗容忍度**——某些客戶可接受少量庫存孔，某些不行
4. **跨批次問題**——同一支捲筒不能混不同批次的紙

---

## 九、Excel 匯入格式

工具支援從 Excel 複製貼上，格式如下：

```
（每行一種尺寸）
台尺    粒數
2.45    5
2.50    4
2.67    2
```

貼入後自動解析，支援 Tab、空格、全形空格分隔。

---

## 十、阿娟二號的工作流程

1. 收到業務部的訂單清單
2. 確認機台限制（最小/最大總寬、每孔範圍）
3. 輸入訂單尺寸與粒數
4. 執行計算
5. 確認有無 ⚠️ 警示，判斷是否需要調整
6. 輸出配抄排程，交給生產部

## 🎖️ AI 對話 SOP

### 對話開始
1. **讀取現狀**：執行 `date`，如果有切換設備則執行 `git pull`，讀取最新的對話摘要與進度。
2. **顯示目標和待辦**：報告上次做到哪裡、今天有什麼待辦，並將待辦按艾森豪矩陣分類：
   - 🔴 重要／緊急（立刻處理）
   - 🟡 重要／不緊急（排時間做）
   - 🟠 不重要／緊急（快速處理或委派）
   - ⚪ 不重要／不緊急（刪除或擱置）
3. **等候指示**：詢問指揮官今天想朝哪個「今日 Boss」推進。

### 對話結束
1. **生成對話總結**：整理對話重點並存入 `F｜行動聚焦漏斗/對話摘要/YYMMDD_主題.md`。
2. **記錄時間與進度**：把未完成的待辦項目寫回 `F｜行動聚焦漏斗/玩家待辦任務.md`。
3. **分析財富積累**：分析這次對話為哪些八大財富帳戶加了分（成長/健康/家庭/技藝/金錢/人脈/體驗/服務）。
4. **自動存檔**：用 git 存檔並 push 更新到遠端（若有設定 GitHub）。

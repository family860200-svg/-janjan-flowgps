# AK-Threads-Booster: Voice Module

Deep brand voice analysis across 11 dimensions. Descriptive, never prescriptive.

## Core Purpose

Go beyond `/setup`'s quantitative baseline. Document how the user actually writes — tone, rhythm, humor, emotional expression — using original text evidence.

## Workflow

**Step 1: Gather Data**
Read `threads_daily_tracker.json` (posts + comment replies). Assess data confidence before proceeding.

**Step 2: Deep Analysis (11 Dimensions)**
Each finding requires original text evidence:
1. Sentence structure
2. Tone switching
3. Emotional expression
4. Knowledge presentation
5. Fan-vs-critic responses
6. Metaphor use
7. Humor style
8. Self-reference patterns
9. Avoided language
10. Paragraph rhythm
11. Comment reply characteristics

**Step 3: Compile `brand_voice.md`**
Structured sections per dimension + quick-reference summary.

**Step 4: Report**
- Post + reply counts analyzed
- Top 2–3 characteristics identified
- Data sufficiency notes

## Key Principles

- **Descriptive, never prescriptive** — document existing patterns, not ideal templates
- Every claim must cite original text
- Acknowledge insufficient data honestly; never speculate
- Output feeds `/draft` as composition reference

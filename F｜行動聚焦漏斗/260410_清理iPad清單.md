# iPad 清理清單

> 建立時間：2026-04-10

---

## 📦 儲存空間清理

- [ ] 設定 → 一般 → iPad 儲存空間，確認佔用最多的 App
- [ ] 刪除長期未用的 App（超過 3 個月沒開）
- [ ] 清空照片「最近刪除」相簿
- [ ] 刪除重複或模糊的照片（建議用「Google Photos」自動偵測）
- [ ] 清理各 App 的快取（LINE、YouTube、瀏覽器）
- [ ] 卸載大型遊戲或影片 App（若已不使用）

---

## 📱 App 整理

- [ ] 將常用 App 移至第一頁
- [ ] 建立資料夾分類（工作 / 創作 / 生活 / 學習）
- [ ] 刪除重複功能的 App（只留一個筆記 App、一個行事曆）
- [ ] 關閉不需要的 App 通知（設定 → 通知）

---

## ☁️ 雲端 & 備份

- [ ] 確認 iCloud 備份是否開啟（設定 → Apple ID → iCloud → iCloud 備份）
- [ ] 手動執行一次備份
- [ ] 檢查 iCloud 容量是否足夠

---

## 🔒 安全 & 帳號

- [ ] 確認已登出不再使用的帳號（設定 → 隱私權）
- [ ] 更新 iPad 系統版本（設定 → 一般 → 軟體更新）
- [ ] 確認 Apple ID 雙重認證已開啟

---

## 🧹 裝置實體清潔

- [ ] 用乾布或專用螢幕布擦拭螢幕
- [ ] 清理充電孔灰塵（用牙籤或小刷子）
- [ ] 清潔保護殼與螢幕保護貼邊緣

---

## ✅ 完成後

- 儲存空間應釋出至少 10%
- App 排列整齊，找起來輕鬆
- 備份完成，心安理得

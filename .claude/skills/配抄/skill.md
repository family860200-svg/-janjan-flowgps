# 配抄工具 Skill（/配抄）

> 說「/配抄」或「打開配抄」即自動執行

---

## 觸發方式

- `/配抄` — 詢問版本後開啟
- `/配抄 台尺` — 直接開台尺版
- `/配抄 英吋` — 直接開英吋版
- `/配抄 MM` — 直接開 MM 版
- `/配抄 公分` — 直接開公分版

---

## 執行流程

### Step 1：判斷版本

如果使用者有指定版本，直接跳 Step 2。
若未指定，詢問：「台尺、英吋、MM、公分，哪個版本？」

版本對應檔案：
| 版本 | 檔案 |
|------|------|
| 台尺 | `L｜資源放大槓桿/公司配抄專用/PM2配抄工具.html` |
| 英吋 | `L｜資源放大槓桿/公司配抄專用/PM2配抄工具_英吋.html` |
| MM   | `L｜資源放大槓桿/公司配抄專用/PM2配抄工具_MM.html` |
| 公分 | `L｜資源放大槓桿/公司配抄專用/PM2配抄工具_公分.html` |

### Step 2：開啟工具

執行：
```bash
powershell.exe -Command "Invoke-Item 'C:\Users\user\Documents\JANJAN x FlowGPS\L｜資源放大槓桿\公司配抄專用\PM2配抄工具{版本}.html'"
```

### Step 3：待命

工具開啟後說：「{版本}版已開啟，把訂單數據給我，一起來算。」

---

## 注意事項

- 公尺單位 → 建議換算成 MM（×1000）使用 MM 版
- 台尺工具已優化：庫存填補為紅色、排在孔2／孔3
